> # E-commerce_Platform_DB
이커머스 플랫폼 데이터베이스 구축 프로젝트 
<br/>
<br/>

> ## 🖥 프로젝트 요약
- E-commerce 데이터를 기반으로 고객 행동을 분석하기 위해 진행한 프로젝트
- 효율적인 데이터 관리를 위한 SQL 테이블 구축
- 고객의 행동 패턴을 이해하기 위한 분석 및 Tableau 시각화
